Getting Started

Copy-paste the stylesheet and javascript link into your <head> to load our CSS and functions. 

          
          <link rel="stylesheet" href="https://res-tracker.herokuapp.com/lib/resTracker.css"/>
          

          <script type="text/javascript" src="https://res-tracker.herokuapp.com/lib/resTracker.js"></>

resTracker.js Object

All functions from resTracker were required to call from the resTracker.js object, the library base object can be instantiated as

          
            const f = new resTracker();

Usage

Any functions should be called on this object for example

          
            const table = f.generateReport(exampleData, "report1", true);

generateReport(data, id, sub)

generateReport: function(data, id, sub) needs developers to provide order summary object data, a string id, and an optional boolean sub. The function is used to generate order summary tables, with sub being true, the table will show all the subtotal information for an order object such as tax, tip, and list of dishes object information , else, the table will only show the total amount.

editReportDetail(reportId, editInfo, mark)

editReportDetail: function(reportId, editInfo, mark) needs developers to provide the table id that need to modify on, as well as the editInfo which varies in format. If the developer is modifying dish objects in one order, editInfo contains the order number, and a new dish object to change to. If the developer is modifying time, date, or restaurant name, editInfo contains the field id to be changed, and string or time object that changes to. Mark is optional for developers to highlight the modified fields.

the existing report id that need to be modified on

addOrder(data, reportId, mark, currentTime)

addOrder: function(data, reportId, mark, currentTime) needs developer to input a new order object to append to the table with id “reportId”. Mark is optional for developers to highlight the modified fields, and currentTime is optional for developers to update the new order time to current EDT.
