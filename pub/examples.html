Generate Report

Creating a resTracker object

All functions from resTracker were required to call from the resTracker.js object, the library base object can be instantiated as

          
    const f = new resTracker();

Generate Dish Summary

Creating a dish summary object

Defining a dish summary object as below, developers can use the below function to generate a dish summary table, note that the manually defined exampleDSData is not exported from report example data, so the content information does not match.

To define a dish summary object representing the report data, use exportReport function

            
    const exampleDSData = {
        "partners": ["DoorDash", "Uber Eat", "Skip The Dishes"],
        "dishes": [
            {
                "name": "Small Pepperoni", "price": "25", "count": "9"
            },
            {
                "name": "Large Pepperoni", "price": "35", "count": "2"
            },
            {
                "name": "Small Hawaiian", "price": "18", "count": "2"
            },
            {
                "name": "Small Canadian", "price": "15", "count": "1"
            }
        ]
    }

With the sample data described above, the second example generates a dish summary with "ds1" as the summary id, with sort functionality and set filter to false

  • JavaScript
  • HTML 
                        
    < div id="example2" >
    < /div >
                        
                        
                    
                        
    const table = f.generateDishSummary(exampleDSData, "ds1", false, true);
    document.getElementById("example2").appendChild(table)

Modify Report Information

Changing restaurant name

With the given edit infomations for report1, name of the restaurant are changed with mark being true

  • JavaScript
  • HTML 
                        
    < div id="example3" >
    < /div >
                        
                        
                    
                        
        const table = f.generateReport(exampleData, "report1", true);
        document.getElementById("example3").appendChild(table)
    
        const editInfo1 = ["title", "restaurant", "Pizza Hut"];
        f.editReportDetail("report1", editInfo1, true);

Changing order to current time or specific time

With the given edit infomations for report1, time for orders are changed accordingly

As shown below, the time for order is being changed into the current EDT time, and order 2 time being set to 12:59:00

  • JavaScript
  • HTML 
                        
    < div id="example4" >
    < /div >
                        
                        
                    
                        
    const table = f.generateReport(exampleData, "report1", true);
    document.getElementById("example4").appendChild(table)

    const editInfo2 = ["order", 1, ["time", "", true]]
    f.editReportDetail("report1", editInfo2);

    const editInfo3 = ["order", 2, ["time", "12:59:00", false]]
    f.editReportDetail("report1", editInfo3);

Updating tip, tax, partner or ordered dishes

The first edit information indicates the report changes order 1's partner to Skip The Dishes, the next information allows changing $10 tips for order 3, and lastly order 1 had additional dishes replaced the old dish 1.

The order total and revenue of the day are updated correspondingly

  • JavaScript
  • HTML 
                        
    < div id="example5" >
    < /div >
                        
                        
                    
                        
    const table = f.generateReport(exampleData, "report1", true);
    document.getElementById("example5").appendChild(table)

    f.editReportDetail("report1", editInfo5);

Modify Report Orders

Add orders and dishes

addOrder is designed for developers to add order to the given report, developer can further choose "marked" tag or "current-time" auguments optionally for their convinence. In this example, we have a newOrder object as follow.

addDish adds dishes with price and count to a specific order developer put in paramter, in the example, the added 3 small pepperoni pizza to order 5.
Both functions update revenue and subtotals respectively with the changes.

In this example, the order are added at current EDT time

  • JavaScript
  • HTML 
                        
    < div id="example6" >
    < /div >
                        
                        
                    
                        
    const table = f.generateReport(exampleData, "report1", true);
    document.getElementById("example6").appendChild(table)
    
    const newOrder = {
        "orderNum": "5",
        "date": "1 March, 2022",
        "dishes": [{"name": "Large Pepperoni", "price": "35", "count": "2"}],
        "tax": "8.75",
        "tip": "5",
        "partner": "Uber Eat",
    }

    const newDish = {"name": "Small Pepperoni", "price": "25", "count": "3"};

    f.addOrder(newOrder, "report1", false, true);
    f.addDish(newDish, "report1", 5, false);

Remove orders and dishes

In the following example, small canadian from order 1 is removed, thus leaving only small pepperoni left for order 1

Since order 3 only contains small pepperoni, the order 3 is completely removed from the report

The last line represents order 2 is removed from report1

Both functions update revenue and subtotals respectively with the changes.

  • JavaScript
  • HTML 
                        
    < div id="example7" >
    < /div >
                        
                        
                    
                        
    const table = f.generateReport(exampleData, "report1", true);
    document.getElementById("example7").appendChild(table)

    f.removeDish("report1", "1", "Small Canadian");

    f.removeDish("report1", "3", "Small Pepperoni");

    f.removeOrder("report1", "2");

Export Report to Dish Summary

As described in documentation, the report to summary function can export both to object or to DOM representation of the object. For this demonstration, we will show the report exports to dish summary bar graph

The export to dish summary table option is demonstrated in "Combine Everything Together" section

  • JavaScript
  • HTML 
                        
    < div id="example8" >
    < /div >
                        
                        
                    
                        
    const table = f.generateReport(exampleData, "report1", true);
    document.getElementById("example8").appendChild(table)

    const ds = f.exportReport("report1", false, "chart");
	document.getElementById("example8").appendChild(ds);

Export Report to Charts

The export to chart function allows developers to export to 4 different graphs to visualization, the example will be demonstrated with default "all" options, which allows selecting any of the chart type in dropdown

  • JavaScript
  • HTML 
                        
    < div id="example9" >
    < /div >
                        
                        
                    
                        
    const table = f.generateReport(exampleData, "report1", true);
    document.getElementById("example9").appendChild(table)

    const chart = f.exportChart("report1", "all"); 
    //default time interval is 8am to 21pm
    document.getElementById("example9").appendChild(chart);

Combine Everything Together

Notice that in the previous examples, each example have tables/report are generated from exampleData, to give better instuition on what a specific function can do.

Now, we will combine everything together and export the report for charts for visualization

Play around the interactions for chart types, sort and filter abilities

  • JavaScript
  • HTML 
                        
    < div id="example10" >
    < /div >
    < div id="example11" >
    < /div >
                        
                        
                    
                        
    const table = f.generateReport(exampleData, "report1", true);
    document.getElementById("example10").appendChild(table)

    const editInfo1 = ["order", 1, ["time", "", true]]
    f.editReportDetail("report1", editInfo1);

    const editInfo2 = ["order", 2, ["time", "12:59:00", false]]

    f.editReportDetail("report1", editInfo3);
