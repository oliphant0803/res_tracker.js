With the sample data described above, the first example generates a non-editable order summary of the given date, developers can choose the report id for their convinent, and the following table contains subtotal for each order as subtotal augument is true for this example

        
          const f = new resTracker();
          const table = f.generateReport(exampleData, "report1", true);
	        document.getElementById("example1").appendChild(table);

The editReportDetail function allows the developer to modify and update any field of the generated report The below example shows the developer changing the date with "marked" flag indicating the field is modified.

Developers can also change the time to current EDT time or the time they wish to specified for each order, the example below shows the code to change order 1's time to current EDT, and to change order 2's time to 12:59:00 

        
          const editInfo1 = ["title", "date", "2 March, 2022"];
          f.editReportDetail("report1", editInfo1, true);

          const editInfo2 = ["order", 1, ["time", "", true]]
          f.editReportDetail("report1", editInfo2);

          const editInfo3 = ["order", 2, ["time", "12:59:00", false]]
          f.editReportDetail("report1", editInfo3);

editReportDetail is also capable of updating the tip, tax, as well as the details for each ordered dishes, the following code demonstrates a developer may update the tips for order 3 and dish for order 1, the total amount and revenue are be changed associated with the price change. 

        
          const editInfo4 = ["order", 3, ["tip", "10.00"]]
          f.editReportDetail("report1", editInfo4);

          const editInfo5 = ["order", 1, ["dish1", ["Large Pepperoni", "35", "2"]]]
          f.editReportDetail("report1", editInfo5);

addOrder is designed for developers to add order to the given report, developer can further choose "marked" tag or "current-time" auguments optionally for their convinence. In this example, we have a newOrder object as follow. 

        
          const newOrder = {
            "orderNum": "4",
            "dishes": [{"name": "Large Pepperoni", "price": "35", "count": "2"}],
            "tax": "8.75",
            "tip": "5",
            "partner": "Uber Eat",
        }

addDish adds dishes with price and count to a specific order developer put in paramter, in the example, the added 3 small pepperoni pizza to order 4.

Both functions update revenue and subtotals respectively with the changes. 

        
          f.addOrder(newOrder, "report1", false, true);

          f.addDish({"name": "Small Pepperoni", "price": "25", "count": "3"}, "report1", 4, false);
